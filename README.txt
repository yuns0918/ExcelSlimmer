ExcelSlimmer README
===================

ExcelSlimmer.exe 간단 사용 안내입니다.

------------------------------------------------------------
1. 프로그램 개요
------------------------------------------------------------

- ExcelSlimmer는 엑셀 파일(.xlsx, .xlsm)의
  용량을 줄이고, 불필요한 정의/숨은 데이터를 일부 정리해 주는 도구입니다.
- 다음 작업을 선택적으로 실행할 수 있습니다.
  - 이름 정의 정리 (definedNames 정리)
  - 워크시트/도형에 포함된 이미지 리사이즈 및 압축
  - 정밀 슬리머(고급 옵션)로 추가 정리

------------------------------------------------------------
2. 배포 파일 구성
------------------------------------------------------------

- 직원에게 제공되는 파일은 보통 다음 한 개입니다.
  - ExcelSlimmer.exe  (실행 파일)
- 별도의 설치는 필요 없으며, 파일을 원하는 폴더에 복사한 뒤
  더블 클릭해서 실행하면 됩니다.

------------------------------------------------------------
3. 기본 사용 방법
------------------------------------------------------------

1) ExcelSlimmer.exe 실행
   - 프로그램이 열리면 상단의 "슬리머 실행" 탭에서 작업을 진행합니다.

2) 대상 파일 선택
   - "대상 파일" 영역의 "찾기..." 버튼을 눌러 용량을 줄이고 싶은
     엑셀 파일(.xlsx, .xlsm)을 선택합니다.

3) 실행할 기능 선택
   - "이름 정의 정리" : 사용되지 않는 이름 정의(definedNames)를 정리합니다.
   - "이미지 최적화" : 시트에 포함된 이미지의 최대 해상도/품질을 낮춰
     파일 용량을 줄입니다.
     (권장 값은 화면에 표시된 기본값을 사용하면 됩니다.)
   - "정밀 슬리머" : 보다 공격적으로 숨은 XML 데이터 등을
     정리합니다. 일부 파일에서 엑셀이 "복구" 안내를 띄울 수 있으므로
     중요한 문서에는 사용 전 백업을 권장합니다.

4) 정밀 슬리머 옵션 (선택 사항)
   - "정밀 슬리머" 체크를 켜면 아래 옵션들을 선택할 수 있습니다.
     - XML 정리 (calcChain, printerSettings 등)
     - 숨은 XML 데이터 삭제 (customXml 등, 고급 옵션)
     - 이미지 포맷 변경 (PNG→JPG) + 참조 동기화 (고급)
   - 문서 중요도가 높을수록 최소 옵션만 사용하거나, 먼저 테스트 파일로
     결과를 확인한 뒤 적용하는 것을 권장합니다.

5) 실행
   - "슬리머 실행" 버튼을 누르면 진행 상태와 로그가 화면에 표시됩니다.
   - 작업이 완료되면 상태가 "모든 작업 완료"로 바뀝니다.

6) 결과 파일 확인
   - 기본적으로 원본 파일과 같은 폴더에 결과 파일이 생성됩니다.
   - 파일 이름 예시:
     - 원본:   Report.xlsx
     - 결과:   Report_complete.xlsx (필요 시 (1), (2) 번호가 붙을 수 있음)
   - "환경 설정" 탭에서 "완성본 저장 폴더"를 지정하면, 결과 파일을
     항상 지정한 폴더로 모아서 저장할 수도 있습니다.

------------------------------------------------------------
4. 백업 및 안전하게 사용하는 방법
------------------------------------------------------------

- 중요한 문서에 적용하기 전에, 반드시 테스트용 파일에서 먼저 실행해 보세요.
- 가능하면 원본 파일은 회사 공용 저장소(예: 팀 공유 폴더)에 한 번 더
  복사해 두고 사용하면 안전합니다.
- "환경 설정" 탭의 "완성본 저장 폴더에 백업 파일 저장" 옵션을 켜면,
  정리 과정에서 생성되는 백업 파일을 함께 보관할 수 있습니다.
- "정밀 슬리머" 및 그 옵션은 엑셀 내부 구조를 적극적으로 정리하는 기능입니다.
  - 작업 후 엑셀이 "복구" 메시지를 띄울 수 있습니다.
  - 실제 내용에 영향이 없는 경우가 대부분이지만, 반드시 결과를 한 번 열어
    정상적으로 동작하는지 확인해 주세요.

------------------------------------------------------------
5. 자주 묻는 질문(FAQ)
------------------------------------------------------------

Q1. 설치가 필요한가요?
 - 아니요. ExcelSlimmer.exe 파일만 있으면 바로 실행할 수 있습니다.

Q2. 어떤 파일 형식을 지원하나요?
 - 주로 .xlsx, .xlsm 형식을 대상으로 설계되었습니다.
   (암호가 걸린 파일이나 공유 편집 중인 파일은 먼저 닫고 사용해 주세요.)

Q3. 원본 파일이 사라지지는 않나요?
 - 기본 동작은 원본과 다른 이름(예: *_complete.xlsx)으로 결과 파일을 만들어
   저장합니다. 그래도 중요한 문서는 직접 한 번 더 백업해 두는 것을 권장합니다.

Q4. 오류가 발생하거나 결과가 이상한 경우에는?
 - 프로그램을 종료한 뒤, 원본 파일을 다시 열어서 이상이 없는지 확인해 주세요.
 - 문제가 반복되면, 이 도구를 배포한 담당자에게 상황(원본 파일 종류,
   사용한 옵션, 오류 메시지 등)을 알려 주세요.
